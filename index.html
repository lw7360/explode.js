How to Explode

In 3 easy steps.

1. Grab explode.js and jQuery. 
<script src="./explode.js"></script>
<script src="./jquery.js"></script>
Protip, you can use rawgit to include explode.js without downloading it.
2. Find something to explode. 
<div id="explode-me">Explode Me!</div>
Explode Me!
3. Explode it. 
<script>
    explode($("#explode-me"));
</script>
Checkout the README for tons more ways to explode.
